# üöÄ Habit Tracker Pro

A sleek, high-performance Habit Tracking Dashboard built with **React**, **Tailwind CSS**, and **Firebase**. This app features a modern dark-themed "Glassmorphism" UI, real-time data synchronization, and detailed progress analytics.

---

## ‚ú® Key Features

- **User-Specific Storage**  
  Securely stores habits and completion data under individual user nodes in Firebase Realtime Database.

- **Dynamic Habit Table**  
  A sticky, horizontal-scrolling grid that allows users to mark completions for the current month.

- **Intelligent Streaks**  
  Automatically calculates current consecutive completion streaks with a visual "Fire" indicator.

- **Advanced Analytics**  
  - **Monthly Performance:** Smooth line chart showing overall daily completion counts across all habits.  
  - **Individual Habit Analysis:** Detailed bar charts for specific habits, including success rate percentages.  
  - **Monthly Summary:** High-level insights identifying your "MVP Habit" and your biggest "Challenge Habit."

- **Real-time Notifications**  
  Integrated with React Hot Toast for sleek, non-intrusive feedback on user actions.

- **Responsive Design**  
  Fully optimized for mobile, tablet, and desktop views using Tailwind's grid system.
